# Activity 2 - Week 2 Overviews
This activity is a compound interest calculator that takes initial savings amount, years of saving, number of compounding periods and the annual interest rate.
For each year of savings it returns the total savings for that year. it also prints out how many years it would take for the savings amount to double

Skills: Learnt how to handle different data types, use mathematical operaters and use functions
